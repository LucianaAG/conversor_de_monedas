¡ACLARACION!: El proyecto se encuentra en la carpeta src/src

Este proyecto es un conversor de monedas en Java que utiliza la ExchangeRate-API para obtener tasas de conversión en tiempo real.
El usuario ingresa:

Moneda base (ej: ARS)

Moneda objetivo (ej: USD)

Cantidad a convertir

El programa realiza la consulta a la API y muestra:

Tasa de conversión

Resultado final

Datos representados en una clase Moneda

Tecnologías utilizadas

Java 17+

Gson

HttpClient (Java)

ExchangeRate-API
